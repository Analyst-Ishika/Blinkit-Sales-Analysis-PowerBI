# Blinkit Sales Analysis Dashboard

## Project Overview
An interactive Power BI dashboard analyzing sales performance, outlet distribution, and item visibility for Blinkit (India's Last Minute App).

## Key Insights
* **Total Revenue:** $1.20M with an average rating of 3.9.
* **Top Performers:** Fruits and Snacks lead the item types in revenue.
* **Outlet Strategy:** Tier 3 locations are driving the highest sales ($472K) compared to Tier 1 and 2.
* **Fat Content:** Regular products ($776K) significantly outperform Low Fat products.

## Tools Used
* **Power BI:** Data visualization and DAX modeling.
* **Power Query:** Data cleaning and transformation.
* **Excel:** Initial data source.
